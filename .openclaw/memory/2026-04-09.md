## Architecture Notes
- **Data flow**: GameSession → GameController.dispatch(GameAction) → emit → ClientState → MapScene (Phaser) + GameShell (React)
- **Three state layers**: GameSession (authoritative), GameController (UI state), React local (presentation)
- **ClientState**: single snapshot consumed by all React components + Phaser renderer
- **Combat animation**: two-phase async — pre-resolve in GameSession, animate in Phaser via pending combat listener
